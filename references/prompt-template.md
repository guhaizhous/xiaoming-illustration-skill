# 生图提示词模板

## 标准单张图提示词

```
16:9横版中文正文配图。纯白背景，大量留白。黑色手绘线稿，细线轻微抖动。
唯一蓝色点缀 #3B82F6，落在[最重要的元素]上。
小明（黑色实心小怪物，蓝色圆点眼，细腿，表情冷静认真）在画面[描述核心动作]。
画面约40%主体，60%留白。
右下角少量蓝色中文手写批注：[1-2个关键词]。
禁止：PPT风格、商业插画、幼稚可爱、复杂架构、左上角标题。
风格：witty, restrained, intelligent。
```

## 场景化模板

### 文章观点图

```
16:9横版中文正文配图。纯白背景。
[核心观点]是这个图的主题。
小明在[做什么动作]，体现[观点的核心含义]。
蓝色点缀在[最有意义的元素]。
右下角批注：[观点关键词]。
```

### 绘本场景图

```
16:9横版绘本风格插图。纯白背景。
场景：[描述场景]
小明作为主角在[做什么]。
蓝色点缀在[场景重点]。
手绘线条，怪诞但清爽。
```

### 知识点拆解图

```
16:9横版解释图。纯白背景。
[知识点名称]被拆解成[核心结构]。
小明在[拆解/重组/操作]这个结构。
蓝色点缀在[关键节点]。
简化的视觉隐喻。
```

### 教学示意图

```
16:9横版教学配图。纯白背景。
[教学概念]用[物理隐喻]解释。
小明在[演示/操作/体验]这个概念。
蓝色点缀标注[重点]。
清晰但不枯燥。
```

## 负面清单（禁止出现）

- 流程框+箭头
- 饼图/柱状图/折线图
- 笑脸/哭脸/表情包
- 闪亮特效/渐变
- 纸纹/网格背景
- "流程图"/"架构图"/"结构图"等标题
- 复杂阴影/3D效果
