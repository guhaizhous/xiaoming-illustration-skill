# QA 检查清单

生成后逐项检查，有问题则重生成或局部编辑。

## 必须满足

- [ ] 纯白干净背景，无纸纹/渐变/阴影
- [ ] 大量留白，主体占画面40%-60%
- [ ] 黑色手绘线条，轻微抖动感
- [ ] 唯一蓝色点缀 #3B82F6，只在一个元素上
- [ ] 小明是画面核心动作主体
- [ ] 小明参与画面核心动作，不是装饰
- [ ] 中文批注少量（1-3个词），手写感

## 禁止出现

- [ ] 没有PPT风格（无流程框、无箭头链）
- [ ] 没有商业插画感（无扁平、无精致过度）
- [ ] 没有幼稚可爱（无大眼睛、无圆脸）
- [ ] 没有复杂架构图
- [ ] 没有左上角类型标题
- [ ] 没有错字（检查中文批注）
- [ ] 小明不是站在角落看热闹

## 风格判断

如果满足以下任一条件，需要重生成：
- 去掉小明，画面核心意思仍然成立 → 小明太装饰
- 画面太满，没有呼吸空间 → 留白不足
- 像教材/说明书/信息图 → 太过说教
- 小明太可爱/太生动 → 不够克制

## 迭代优先级

1. 先修复小明动作（核心问题）
2. 再修复留白/点缀色
3. 最后调整批注
